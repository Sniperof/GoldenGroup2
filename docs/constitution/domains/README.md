# Golden CRM — Domains Index

> فهرس الدومينات الرئيسية في المشروع.

## الدومينات المعتمدة حاليًا
- `permissions` — الصلاحيات والنطاقات
- `org-structure` — الفروع، المناطق، الأقسام، الموظفون
- `clients` — الزبائن، الملكية، الإسناد، دورة الحياة
- `planning` — الجدولة، التوزيع، التغطية، الأهداف
- `tasks` — المهام التشغيلية والميدانية
- `visits` — الزيارات وأنواعها وحالاتها
- `telemarketing` — الاتصال، الأهداف، الربط التشغيلي
- `contracts` — العقود والدورة المرتبطة بها
- `devices-maintenance` — الأجهزة، الصيانة، قطع الغيار
- `jobs-recruitment` — الشواغر، الطلبات، المقابلات، التدريب
- `admin-settings` — الإعدادات العامة، القوائم، الأدوار، الصلاحيات

## ملاحظات
- كل دومين سيأخذ ملف دستور مستقل.
- أي feature كبيرة داخل الدومين ستأخذ ملفًا خاصًا بها داخل `features/`.
